## 📦 Installation

### Method 1: Install from GitHub Release
1. Visit the [Releases page]({RELEASE_URL})
2. Download the `craftengine-vsc-{VERSION}.vsix` file
3. Open VS Code and press `Ctrl+Shift+P` to open the command palette
4. Type `Extensions: Install from VSIX...`
5. Select the downloaded `.vsix` file to install

### Method 2: Manual Installation
```bash
# Download the extension file
wget {DOWNLOAD_URL}

# Install to VS Code
code --install-extension craftengine-vsc-{VERSION}.vsix
```

## 🔄 Upgrade Instructions

If you have an older version installed, please follow these steps to upgrade:

1. **Uninstall the old version**:
   - Open VS Code Extensions panel (`Ctrl+Shift+X`)
   - Search for "CraftEngine"
   - Click the uninstall button

2. **Install the new version**:
   - Follow the installation methods above
